# SQL & Data — 3-Month Roadmap

> **Starting point:** You already know the basics of SQL.

---

## Overview

| Phase | Topic | Timeline | Milestone |
|-------|-------|----------|-----------|
| 1 | Solidify & Extend | Weeks 1–3 | Write multi-table queries with confidence |
| 2 | Analytical SQL | Weeks 4–7 | Use window functions and CTEs fluently |
| 3 | Performance & Data Modeling | Weeks 8–10 | Design a schema and optimize slow queries |
| 4 | Real-world & Tooling | Weeks 11–12 | Portfolio-ready — SQL + a data tool |

---

## Phase 1 — Solidify & Extend
**Weeks 1–3**

### Topics
- [ ] Review SELECT, WHERE, GROUP BY, ORDER BY
- [ ] Master all JOIN types (INNER, LEFT, RIGHT, FULL)
- [ ] Subqueries & nested SELECT
- [ ] String & date functions
- [ ] CASE WHEN expressions
- [ ] Practice: LeetCode SQL Easy → Medium

### Project
> Query a public dataset (e.g. NYC taxi data) — answer 5 business questions

### Milestone
🎯 Can write multi-table queries with confidence

---

## Phase 2 — Analytical SQL
**Weeks 4–7**

### Topics
- [ ] Window functions: ROW_NUMBER, RANK, DENSE_RANK
- [ ] LEAD / LAG for time-series analysis
- [ ] Aggregates: SUM / AVG OVER (PARTITION BY)
- [ ] CTEs (WITH clause) & recursive CTEs
- [ ] PIVOT / UNPIVOT patterns
- [ ] Practice: Mode Analytics SQL Tutorial

### Project
> Sales analysis — calculate running totals, rank products, detect trends

### Milestone
🎯 Can use window functions and CTEs fluently

---

## Phase 3 — Performance & Data Modeling
**Weeks 8–10**

### Topics
- [ ] Indexes: when & how to use them
- [ ] EXPLAIN / query plan analysis
- [ ] Normalization (1NF → 3NF)
- [ ] Star schema & dimensional modeling
- [ ] Transactions & ACID properties
- [ ] Stored procedures & views

### Project
> Design a star schema for an e-commerce database, load and query it

### Milestone
🎯 Can design a schema and optimize slow queries

---

## Phase 4 — Real-world & Tooling
**Weeks 11–12**

### Topics
- [ ] SQL + Python (pandas `read_sql`, SQLAlchemy)
- [ ] dbt fundamentals (transformations-as-code)
- [ ] BigQuery / Snowflake / Redshift basics
- [ ] Data cleaning patterns in SQL
- [ ] Build a portfolio project end-to-end
- [ ] Mock interview: 3 SQL take-home problems

### Project
> Capstone: Full analysis with SQL + visualization (Tableau, Metabase, or Python pandas)

### Milestone
🎯 Portfolio-ready — SQL + a data tool
